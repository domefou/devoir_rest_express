Module: models/catways

Ce modèle définit la structure des documents `catways` stockés dans une collection MongoDB. Chaque document représente un catway avec des propriétés telles que son identifiant, son numéro, son type, et son état. Le modèle utilise également les timestamps pour inclure automatiquement les champs `createdAt` et `updatedAt`.
Properties:
Name Type Description
catwaysId String Identifiant du catway (facultatif).
catwaysNumber Number Numéro unique du catway (obligatoire).
catwaysType String Type du catway (obligatoire).
catwaysState String État du catway (obligatoire).
createdAt Date Date de création du document (ajoutée automatiquement par Mongoose).
updatedAt Date Date de la dernière mise à jour du document (ajoutée automatiquement par Mongoose).
